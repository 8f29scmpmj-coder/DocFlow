Common questions / 常见问题

English

Where are output files saved?
Output files are written to the folder selected by the user before the workflow starts.

Why can conversion fail?
Conversion may fail if the source PDF is encrypted, damaged, unsupported, or if the output folder permission is no longer valid.

How do I restore a purchase?
Use the restore purchase option inside the app while signed in with the Apple ID used for the original purchase.

Does the app upload my documents?
The app is designed to process selected files locally on the user's Mac.

中文

输出文件保存在哪里？
输出文件会写入用户在工作流开始前选择的目标文件夹。

为什么转换会失败？
如果源 PDF 被加密、已损坏、不受支持，或输出目录权限失效，转换可能失败。

如何恢复购买？
请在应用内使用恢复购买功能，并确保当前登录的是原始购买所使用的 Apple ID。

应用会上传我的文档吗？
应用设计目标是在用户本地 Mac 上处理所选文件。

Issue submission tips / 问题提交建议

English

  • State which workflow you used: conversion, compression, or images to PDF.
  • Describe the expected result and the actual result.
  • Do not attach sensitive documents unless you are comfortable sharing them publicly.

中文

  • 请说明你使用的是哪种工作流：转换、压缩或图片转 PDF。
  • 请描述预期结果与实际结果。
  • 除非你确认可以公开分享，否则不要上传敏感文档作为附件。
