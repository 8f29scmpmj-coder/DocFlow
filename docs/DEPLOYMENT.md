# DocFlow PDF GitHub Pages Deployment

## Publish the site

1. Push the repository to GitHub: `https://github.com/8f29scmpmj-coder/DocFlowPDF`
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: your publishing branch, usually `main`
   - `Folder`: `/docs`
5. Save the settings and wait for GitHub Pages to publish.

GitHub Pages will then provide a public site URL similar to:

- `https://8f29scmpmj-coder.github.io/DocFlowPDF/`
