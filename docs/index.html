Product Summary

What DocFlow PDF does

English

DocFlow PDF is a document utility app for Mac users who need focused PDF workflows without sending files to a remote service. The app helps users convert PDFs into editable or exportable formats, reduce PDF size for sharing, and merge images into a single PDF.

中文

DocFlow PDF 是一款面向 Mac 用户的文档处理工具，专注于本地 PDF 工作流，不依赖将文件上传到远程服务。应用支持 PDF 转换为可编辑或可导出的格式、压缩 PDF 体积，以及将多张图片合并为单个 PDF。

PDF conversion

English

Convert PDF files into Word, Excel, PowerPoint, CSV, text, Markdown, HTML, images, OFD, and searchable PDF depending on the selected workflow.

中文

根据所选工作流，支持将 PDF 转换为 Word、Excel、PowerPoint、CSV、文本、Markdown、HTML、图片、OFD 以及可搜索 PDF。

PDF compression

English

Reduce file size with compression presets designed for common sharing and archive scenarios while trying to keep documents readable.

中文

通过压缩预设降低 PDF 文件体积，适用于常见分享与归档场景，同时尽量保持文档可读性。

Images to PDF

English

Select multiple images, reorder them, and export a single merged PDF directly on macOS.

中文

可选择多张图片、调整顺序，并直接在 macOS 上导出合并后的单个 PDF 文件。

Local-first processing

Document handling model

English

DocFlow PDF is designed to process the files selected by the user on the local Mac environment. The app does not market itself as a cloud storage or document hosting service.

中文

DocFlow PDF 设计目标是在用户本地 Mac 环境中处理所选文件。该应用并非云存储或在线文档托管服务。

Pro Upgrade

Free and Pro positioning

English

The free experience is intended for lighter usage. DocFlow PDF Pro is offered as a one-time lifetime in-app purchase that unlocks broader batch and workflow limits within the same app listing.

  • Batch conversion workflows
  • Folder import in supported flows
  • Larger PDF conversion limits
  • All compression presets
  • Multi-image PDF creation at larger scale

中文

免费版定位于轻量使用场景。DocFlow PDF Pro 作为一次性买断型应用内购买提供，在同一应用条目内解锁更高的批量和工作流能力。

  • 批量转换工作流
  • 部分流程中的文件夹导入能力
  • 更高的 PDF 转换限制
  • 全部压缩预设
  • 更大规模的多图转 PDF 能力
